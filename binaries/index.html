TollGate Binary Downloads

OpenWRT plugin for Bitcoin-powered WiFi captive portal

About TollGate

TollGate enables OpenWRT routers to accept Bitcoin payments for internet access through a captive portal interface.

Build From Source

Clone the repository:

git clone https://github.com/OpenTollGate/TollGateNostrToolKit.git
cd TollGateNostrToolKit/

Run the build coordinator (requires sudo privileges):

./build_coordinator.sh

Note: Initial build may take over an hour. Subsequent builds are faster if only modifying files.

Flashing Instructions

  1. Connect your OpenWRT router to your computer
  2. Find your router's IP address using ifconfig
  3. Copy the sysupgrade file to your router: 
    scp openwrt-ath79-nand-glinet_gl-ar300m-nor-squashfs-sysupgrade_[commit_hash].bin root@[router_ip]:/tmp/.
  4. SSH into your router and navigate to /tmp
  5. Flash the router: 
    sysupgrade -n openwrt-ath79-nand-glinet_gl-ar300m-nor-squashfs-sysupgrade_[commit_hash].bin
  6. Wait for the router to reboot (few minutes)
  7. Find the new IP address using ifconfig
  8. SSH into the router and follow the prompts to set your password and LNURL

Requirements

  • OpenWRT compatible router (tested on GL.iNet GL-AR300M)
  • Linux build environment with sudo privileges
  • Internet connection for downloading dependencies
